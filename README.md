# コマンド類
## 管理者様専用コマンド
```/set``` このコマンドではテキストの反応チャンネルや、画像の質問の制限を設定します

## 通常コマンド
```/ping``` pingです

```/approval``` 履歴の保存設定を設定します

# 使用方法

### ・画像質問の使用方法

画像質問では、送信前に『あいちゃ』をメンション・文章内に『あいちゃ』と呼ぶことで可能です。
また、スマホの場合長押ししてアプリというところの質問からもご利用いただけます。
パソコンの場合はその他からアプリです。

### ・テキスト会話の使用方法

テキストでの会話では、『あいちゃ』を文章内に入れる、またはメンションをすることに入れることで会話が可能です

### ・セッション機能について

会話の最初にセッション機能(履歴)を許可するかの問いがあります。
許可をした場合、あいちゃの返信をリプライすることで、会話を引きつぐことができます。

また、リプライをせずに会話を利用することで新しいセッションを作ることができます。
最大セッション保持数は5件です。
6件目の新しい会話を開始すると、一番古いセッションから自動的に削除されます。

# データの保存について

### ・第一条
このbotで取得した会話内容などは、本人の同意なしに他人への譲渡を禁ずる

### ・第二条
本botが取得し、保存するデータは、ユーザーによる承認制であり、ユーザーはいつでも履歴の削除を申請することが出来る

# 禁止行為について

### ・第一条
本botを利用するにあたって、過度なリクエストなどは禁止しております。
発見次第使用権原を剥奪する権利をbot側は所有しているものとします。
